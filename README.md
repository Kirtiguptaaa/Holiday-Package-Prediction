# Holiday Package Prediction

A Machine Learning model that predicts whether a customer is likely to purchase a holiday package based on their demographic and behavioral data.

## Overview

This project uses data-driven techniques to analyze customer information and predict purchase behavior. It helps travel companies target the right audience and improve marketing strategies.

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Model training
- Customer purchase prediction
- Model evaluation using multiple metrics
##  Models Tried

- Logistic Regression
- Decision Tree
- Gradient Boosting
- Random Forest (Best Performing)

### Best Model: Random Forest

Random Forest achieved the highest accuracy due to its ability to:
- Capture non-linear relationships
- Reduce overfitting using ensemble learning
- Handle mixed data types effectively

## Tech Stack

- Python

- Scikit-learn

##  Key Insights

- The model achieves high accuracy (93.25%) on unseen data.

- High precision (94.93%) indicates strong ability to correctly identify potential buyers.

- Lower recall (68.95%) suggests some potential customers are missed.
